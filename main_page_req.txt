기업 검색을 완료하고 요약 보고서 생성 버튼을 클릭하는 경우 제공해야하는 기능을 알려줄게.

1. 기업 기본 정보를 테이블 형태로 제공
2. 분석에 사용한 뉴스 기사 상위 3개를 제목과 언론사 등 기본 정보 표시
3. ai가 판단한 적정주가와, 현재 주가 비교
4. 최근 1년간 주가 추이 / 이동평균선, RSI등 을 그래프 형식으로 제공(미래 추이도 점선으로 표시하고 예측치라고 주의하는 문구 추가)
5. 재무건정성 유동성 등 주요 지표 표시
6. 생성형 AI의 평가 요약 표시
7. 생성형 AI의 상세 평가 표시

이걸 굳이 한페이지에 다 채워 넣지 않고 새로 스크롤 하는 형식으로 만들어주면 되는데, 적절하게 배치해서 정말 완성도 높은 요약 보고서 화면이 만들어졌으면 좋겠어.

그리고 이후 요약보고서 내용을 토대로 openai에게 간단한 질의를 할 수 있는 채팅창이 표시됐으면 좋겠는데, 이건 PC버전인 경우에만 제공하도록 해줘.

그리고 DART에 보고서 아이디를 파라미터에 넣고 누르면 보고서 원본을 볼 수 있는데 그것도 정기보고서 원본 보기 버튼을 만들어서 제공해줘.
